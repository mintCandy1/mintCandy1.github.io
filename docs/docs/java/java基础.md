## 数据类型
计算机使用0或1的位表示信息，又称为比特bit。8个bit组成一组，称为字节byte。
### 基本数据类型
java有8种基本数据类型：
● byte 1字节
● short 2字节
● int 4字节
● long 8字节
● float 4字节
● double 8字节
● char 2字节
● boolean 4字节，或1字节（boolean数组中）
### 引用类型：对象
将对象赋给一个变量，该变量的值是对象的地址，称作该变量指向对象。
